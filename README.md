# Visitor Pass Management System

## Overview
This is a MERN Stack based Visitor Pass Management System project.

## Features
- Login Page
- Dashboard
- Add Visitor
- Delete Visitor
- MongoDB Atlas Integration
- React Router Navigation

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB Atlas

## Setup Instructions

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
